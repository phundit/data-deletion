Phundit Data Deletion Request

How to Request Data Deletion:

  1. In-App: Go to Settings → Privacy → "Request Data Deletion"
  2. Via Email: Send your request to support@phundit.app with subject "Data Deletion Request"

What We Delete:

  • Personal information (name, email, phone number)
  • Transaction history
  • Investment preferences
  • App activity data

What We Retain (For Legal Compliance):

  • Financial transaction records (required for 5 years by Ghanaian law)
  • Fraud prevention data

Requests are processed within 14 business days. You'll receive confirmation via email.
